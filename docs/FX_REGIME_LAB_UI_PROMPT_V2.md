## 2. Design brief (summary)

**Intent:** A serious independent macro researcher proving work publicly — *financial research magazine that came alive* (editorial, warm precision, calm confidence). **Not** a crypto dashboard, SaaS product, Bloomberg clone, or student portfolio.

**Theme:** **Light** — page `#F4F5F3`, cards `#FFFFFF`, **dark contrast islands** `#0f1420` for live data blocks. Accent navy `#1B3A6B`. Pair colours: EUR/USD `#2563a8`, USD/JPY `#b86b2a`, USD/INR `#a63030`.

**Background:** Full-viewport **HTML5 Canvas** behind content (`pointer-events: none`), `requestAnimationFrame`, **no libraries** — Layer 1 slow pseudo price lines (pair colours, ~6–8% opacity), Layer 2 sine “pulse” waves (navy, subtle), Layer 3 sparse 2px dots (pair colours, ~1/2s spawn). **Mobile:** layer 1 only, ~50% opacity.

**Nav:** Sticky 68px, `backdrop-filter: blur(12px)` on **nav only** (not cards). Centered links on desktop; hamburger + full-screen overlay on mobile.

**Motion allowed:** Canvas; count-up on hero / scroll; fade+translate on scroll; card hover ≤2px lift; pipeline dot pulse. **Not allowed:** parallax (beyond canvas), 3D, gradient animations, scroll hijack, confetti, hero skeletons.

**Components:** Ghost buttons only; regime badges; light + dark signal cards; validation table styling per CSS in `site/assets/site.css`.

**Pages:** `/` landing (7 sections), `/dashboard/`, `/brief/` hub + `/brief/latest.html`, `/performance/`, `/about/`, `/newsletter` redirect.

## 3. Conflict resolutions (vs other docs)

| Conflict | Resolution |
|----------|------------|
| Cursor rule “dark theme always” for product | **Exception:** public **`site/`** uses **v2 light** editorial UI. **Plotly brief** (`latest.html`) may remain **dark** until `create_html_brief.py` / `static/styles.css` are restyled. Pipeline charts and GitHub Pages `index.html` unchanged. |
| “No particles” vs canvas dots | Dots are **sparse ambient ticks**, not particle FX / confetti. |
| Glassmorphism | **Nav blur only**; no `backdrop-filter` on cards. |
| Stitch v1 (Sora / dark navy-black) | **Deprecated** for `site/`; use v2 tokens only. |
